Help

What it does: Applies a double-stroke style and optional boxed directional arrows to GPX/SVG tracks directly in your browser.

  1. Upload your .svg file.
  2. Choose outer, inner, and optional arrow color.
  3. Adjust stroke factors.
  4. Set the Group ID (default trk1).
  5. Optionally remove watermark text.
  6. Optionally keep only the essential SVG map content.
  7. Enable boxed directional arrows if needed.
  8. Click Download SVG to save the result.
